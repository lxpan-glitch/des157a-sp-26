# Portal Page - DES 157A

## Overview
This is my portal page for DES 157A, Winter 2025. It’s designed to be a clean, organized hub where you can find all my projects for this quarter, including Research, MadLibs, Every Picture, and Game On.

## My Design Choices
* **The Grid**: I went with a 2x2 grid because it keeps everything aligned and easy to read. I wanted a look that felt solid and professional.
* **Style & Effects**: 
    * I used a grayscale filter on all the project images to make them look consistent, even though the original photos were all different colors.
    * I added a hover effect so the cards lift up when you mouse over them—it just makes the page feel more interactive.
    * There's also a small floating animation on the computer illustration in the banner.
* **Typography**: I used 'Holtwood One SC' for the big titles and 'Lalezar' for the links to give it a unique, bold personality.

## Technical Stuff
* Used `* { margin: 0; }` in CSS to make sure there are no weird gaps around the edges of the page.
* Used `object-fit: cover` for the project images so they fill the boxes perfectly without getting stretched.
* Encoded my email address in the HTML to follow the security requirements from the assignment.
